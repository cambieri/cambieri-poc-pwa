PWA POC

Relevant point of attention:

  • index.html
    • only change the title
  • app.component
    • remove everything about Platform, SplashScreen, StatusBar
    • the initializeApp function
  • internal pages
    • nothing to do
  • ngsw-config.json
    • add "/manifest.webmanifest" in app section
    • add ONLY THE USED SVG in assets section!!! In this case:
      • "/svg/ios-flag.svg"
      • "/svg/md-flag.svg"
    • set "installMode": "prefetch" in assets section
  • manifest.webmanifest
    • set "scope" and "start_url" corresponding to the deploy dir (for example: "/poc/pwa/" if the final url is https://develop.cambieri.net/poc/pwa/)
  • VERY IMPORTANT:
    • build with this command "ionic build --prod -- --base-href /poc/pwa/" where the "--base-href" param MUST correspond to the "scope" and "start_url" set in the manifest.webmanifest
    • when you deploy, delete the old files (expecially from root)
    • deploy ONLY THE USED SVG
  • enjoy...
